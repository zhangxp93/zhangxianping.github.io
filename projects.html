测试项目展示

这里展示了我的射频芯片测试项目，涵盖PyVISA自动化、仪器控制和数据分析

射频芯片自动化测试平台

基于Python和PyVISA的射频芯片全自动测试系统，支持多种射频参数测试，包括S参数、噪声系数、功率增益等关键指标。

Python PyVISA NumPy Matplotlib
2024年3月 已完成

多仪器协同控制系统

使用PyVISA统一控制网络分析仪、频谱分析仪、信号发生器等多台仪器，实现复杂射频测试序列的自动化执行。

PyVISA SCPI Threading PyQt5
2024年1月 已完成

射频测试数据分析工具

专业的射频测试数据处理和可视化工具，支持S参数分析、史密斯圆图生成、频域时域变换等功能。

Python SciPy Matplotlib Pandas
2023年11月 已完成

自动化测试报告生成器

基于Python的测试报告自动生成工具，支持多种数据格式导入，生成专业的PDF测试报告和数据图表。

Python ReportLab Jinja2 Openpyxl
2023年9月 维护中

射频仪器自动校准系统

针对网络分析仪和信号发生器的自动校准程序，通过PyVISA实现多台仪器的批量校准和统一管理。

PyVISA SQLite Tkinter ConfigParser
2023年7月 已完成

通用SCPI命令库

基于Python的SCPI命令封装库，支持主流射频仪器厂商的仪器控制，提供统一的API接口和错误处理机制。

Python PyVISA Setuptools Sphinx
2024年2月 开发中
